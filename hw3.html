HTML and Style

The Box Model

HTML and CSS use a box model to position elements. Each element takes up a certain amount of space called a box. There are four parts of the box model:

  1. Content
  2. Padding
  3. Border
  4. Margin

Content

Content is the stuff that goes inside the element. It's the 'main course', things like text or pictures.

Padding

Padding is the space between the content and the rest of the box. If you put a ball in a cardboard box, the ball would be the content, and the air between the ball and the box would be the padding.

Border

The border is the line that seperates the inside of the box from everything else. It would be like the cardboard in our box analogy.

Margin

The margin is the space between elements. If we stacked our cardboard boxes next to each other the margin would be the air between the boxes.

Positioning

Relative Position

Relative position changes the placement of an element based on where it normally would be. Adding to a relative value adds space on that side of the element.

This text is positioned 30px to the right of where it normally would be.

Absoulte Position

Absolute position places an element based on another element.

This paragraph is absolute to a div.

Fixed Position

Fixed position places an element based on the viewport.
